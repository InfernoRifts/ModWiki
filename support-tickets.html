InfernoRifts Support Bot — Quick Guide for Mods

A concise walkthrough for using the Support Bot and handling support tickets on the server. Only staff with the Support role should use the mod features.

1. How Tickets Get Created

  1. Users go to the #support-ticket channel and click the "Open Ticket" button on the embed.
  2. This creates a private ticket channel in the "Unclaimed Tickets" category.
  3. The bot mentions the user with a welcome message and staff-facing buttons.

2. Staff Actions in Tickets

You can use the buttons on the bot's welcome message or the slash commands /claim, /close, and /escalate. All actions should be performed inside ticket channels.

Claim the Ticket

Click the "Claim" button or type /claim. This renames the channel to claimed-<yourname> so others know you're handling it.

Escalate the Ticket

Click "Escalate" or type /escalate. Choose a level (Developer, Moderator, Owner/Manager) from the dropdown, then enter a reason in the modal. The bot will rename the channel and move it to the correct category (for example, dev-tickets).

Close the Ticket

Click "Close" or type /close. You can choose:

+
  • Transcript — Generates .txt and .html logs, sends them to #ticket-logs, and gives you a copy. The ticket stays open so you can review it further.
  • Delete — Deletes the channel after 5 seconds (use this after saving a transcript if you want the physical channel removed).
  • Cancel — Keeps the ticket open.

Tips

  • Tickets are private: only the ticket creator and staff can see the contents.
    • +
  • Test and practice in a dedicated ticket channel — slash commands appear there for staff only.
    • +
  • If something breaks, ping DemonDev with the error details so it can be investigated.
    • +

That's it!

Keep tickets organized and act quickly.
