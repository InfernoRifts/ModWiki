Inventory Rollbacks

Commands to check and create backups with Axir. Click the Copy button to copy a command to your clipboard.

  • /axir view <player>

    Check backups for a player. Shows available backups and metadata (time, size, etc.).

  • /axir save *

    Create a manual backup for all players.

  • /axir save <player>

    Create a manual backup for the specified player.

Examples

/axir save FallenPlayer
/axir FallenPlayer

Notes

  • Make sure you have the required mod permissions before running these commands.
  • When using /axir <player>, you may get a list of available backups — use the backup ID with the rollback command (if applicable).
